JUMPCORE

Project Details:

  • Genre: First-person precision platformer, collectathon, action/puzzle.
  • Platform: PC (Steam / Microsoft Store).
  • Engine: Unity Engine.
  • Team Size: Indie.
  • Roles: Level Designer, Programmer, Environmental Artist.

    • The game started as a personal Unity project of mine in university. After showcasing and pitching the game in various places, a development team was assembled. Since then the project entered Image Campus incubation programm (IC-CUBE) and has been showcase at EVA PLAY 2025.

    Work Example Document and Video:

    You can read in this document a detailed overview of one of the levels I had constructed on a previous build: JUMPCORE - Work Example Document

    Responsibilities:

  • Level Design: Design level layouts and contruct them using Realtime CSG. Lead other level designers. Develop tools that enrich level creation.
  • Programming: Prototype, iterate, and fine-tune game features to deliver engaging player experiences. Script gameplay mechanics, and interactive systems in Unity using C#. The following example is an environment entity that level designers can use to decorate levels.
  • Environment Art: Lighten and texture levels to archive beautiful scenes. Research and give proper references to artist to create props, textures and sprite assets.
